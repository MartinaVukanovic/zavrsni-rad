General steps of Dijkstra's algorithm

1. Choose start node and store child nodes and distances to child nodes

2. Find nearest node based on stored distances

3. For every child node of node picked in previous step find shortest path to start node, continue with node that has shortest path

* Repeat steps 2 and 3 untill all nodes are visited
